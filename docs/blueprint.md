# **App Name**: AI Profit Decoder

## Core Features:

- Profile Aptitude Quiz: A quiz to assess user compatibility with AI-driven income opportunities.
- Real-time Compatibility Analysis: Analyze the quiz results to determine the user's compatibility with AI profit methods, offering immediate feedback and a 'compatibility score'.
- Personalized Offer Display: Based on the compatibility analysis, display a tailored offer with a significant discount.
- Countdown Timer: Implement a countdown timer to create a sense of urgency around the limited-time offer.

## Style Guidelines:

- Background color: Deep black (#050510) to establish a dark mode theme.
- Primary color: Neon green (#39FF14) for buttons and progress bars, conveying action and technological flair.
- Accent color: A slightly lighter, more saturated lime green (#BFFF00) to highlight interactive elements and calls to action. 
- Font: 'Space Grotesk', a modern sans-serif, for headlines and body text, for a techy, clean aesthetic. 
- Mobile-first design prioritizing vertical scrolling, ample whitespace for clarity, and glassmorphism effects on question cards and containers for a frosted glass look.
- Use simple, glowing line icons to represent various steps and features, enhancing the tech-forward appearance.
- Implement subtle neon glow animations around buttons and key elements, with loading animations that simulate data analysis, to engage users and emphasize the AI aspect.
